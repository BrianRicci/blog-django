# Блог на Django

Этот блог не является пет-проектом, т.к. был создан, читая книгу "Django 4 в примерах" Антонио Меле. Это самый первый мой проект на Django. Данный репозиторий создан, чтобы можно было подглядывать и брать что-то отсюда для дальнейших проектов.

Здесь используются:
- Админка
- Маршрутизация
- Представления
- Модели
- Формы
- ORM
- Генерация URL по слагам и генерация самих слагов
- Отправка писем
- Система комментариев
